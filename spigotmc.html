Getting started with SpigotMC

  1. Start by downloading the latest version of SpigotMC below.
  2. Download the starter pack below.
  3. Unzip the file by right clicking it and clicking Extract all.
  4. Copy the Spigot.jar to the folder that you unziped.
  5. Run bash start (Ubuntu) or start.bat (Windows)
